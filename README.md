Installation Guide
------------------
- Extract zip file.
- Place directory called `q2a-SnowX-theme` in qa-theme folder.
- Done

Minimum Question2Answer Version Required
----------------------------------------
- v1.5 (works with v1.7.5, v1.8)

Admin Guide
-----------
- Login with admin id.
- Go to Admin > General page.
- Enable Site theme from admin panel
